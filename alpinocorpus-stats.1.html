NAME

alpinocorpus-stats -- Alpino treebank node statistics

SYNOPSIS

alpinocorpus-stats [options] query treebank ...

DESCRIPTION

The alpinocorpus-stats utility counts the number of occurences of nodes matching query in each Dact treebank. If treebank is a directory, then alpinocorpus-stats will count matching nodes in all treebanks below that directory. The supported query format is XPath 2.0.

The following options are available:

-m MACROFILE
Load macros from MACROFILE. -p
Print relative frequencies.

EXAMPLE

The command:

alpinocorpus-stats '//node[@root="loop"]/@word/string()' lassy.dact

will print the frequency of each word in the treebank that has loop as its root.
