NAME

alpinocorpus-get -- Print Alpino treebank entry to stdout

SYNOPSIS

alpinocorpus-get [options] treebank entry

DESCRIPTION

The alpinocorpus-get utility outputs a treebank entry to stdout. An XPath query can be used to mark nodes.

The following options are available:

-a ATTRIBUTE
Mark nodes selected with 'q' with ATTRIBUTE. -m MACROFILE
Load macros from MACROFILE. -q QUERY
Mark nodes in the output using QUERY (XPath 2.0). -v VALUE
Mark nodes selected with 'q' with VALUE.
