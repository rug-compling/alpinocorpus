NAME

alpinocorpus-xslt -- Apply stylesheet to an Alpino treebank

SYNOPSIS

alpinocorpus-xslt [options] stylesheet treebank ...

DESCRIPTION

The alpinocorpus-xslt utility applies stylesheet to each Dact treebank. If treebank is a directory, then alpinocorpus-xslt will apply stylesheet to each treebank below that directory. stylesheet should be a XSLT stylesheet.

The following options are available:

-g ENTRY
Apply the stylesheet to ENTRY, rather than each entry in the treebank.
-m MACROFILE
Load macros from MACROFILE.
-q QUERY
Filter the treebank using QUERY (XPath 2.0). Nodes in the XML data that match QUERY get the attribute-value pair active=1.
